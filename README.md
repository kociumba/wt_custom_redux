# wt custom redux

wt custom redux is a rewrite and redesign from the ground up of the old and abandoned [wt custom](https://github.com/kociumba/wt_custom) project

it is a utility program, for editing data stored by warthunder in the custom csv format used by the game, 
in general this can be used as a generic visual csv editor, wt custom redux does make some assumptions about the format that are speciific to the warthunder data.
